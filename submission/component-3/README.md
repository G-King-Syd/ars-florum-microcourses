# Component 3 – Frontend Integration (React + API)

## Overview
This component demonstrates client-side integration with the backend API. It allows users to view courses and interact with live data from the server.

## GitHub Repository
https://github.com/G-King-Syd/ars-florum-microcourses

## Features
- Fetch course data from backend API
- Display list of courses dynamically
- Navigate to individual course detail pages
- Update enrolment counts via API (PUT request)

## How to Run

### 1. Navigate to folder
cd submission/component-3

### 2. Install dependencies
npm install

### 3. Start the application
npm start

The app will run at:
http://localhost:3000

## Notes
- Requires Component 2 backend to be running
- Demonstrates full frontend-to-backend integration
