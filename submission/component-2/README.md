# Component 2 – Backend API (Node.js / Express / MongoDB)

## Overview
This component contains the backend implementation for the Ars Florum Micro-Courses platform. It provides API endpoints to manage course data and connects to a MongoDB database.

## GitHub Repository
https://github.com/G-King-Syd/ars-florum-microcourses

## Features
- Express server setup
- MongoDB connection using Mongoose
- Course data model
- API endpoints for:
  - Retrieving courses (GET)
  - Creating courses (POST)
  - Updating enrolments (PUT)
