# Component 1 – Frontend Application (React)

## Overview
This component is a React-based frontend for the **Ars Florum Micro-Courses** platform. It allows users to browse available courses and view course content through a simple and visually structured interface.

## GitHub Repository
https://github.com/G-King-Syd/ars-florum-microcourses

## Features
- Browse a list of micro-courses
- View course images and descriptions
- Navigate between pages using React
- Clean and responsive layout

## How to Run

### 1. Navigate to the folder
cd submission/component-1

### 2. Install dependencies
npm install

### 3. Start the application
npm start

The app will run at:
http://localhost:3000

## Notes
- This component includes **frontend functionality only**
- Backend functionality is implemented in later components
- Images are stored locally in the `images` folder
